# 傲为 ERP 销售中心新框架

本目录是按 `vue3-admin-better` 的后台框架思路重建的销售中心前端骨架。

已转化范围：

- 销售工作台
- 客户管理
- 销售计划
- 报价管理
- 合同管理
- 订单管理

框架能力：

- Vue 3 + Vite + TypeScript
- Element Plus 应用壳
- 顶部栏、侧边菜单、标签页
- Pinia 状态管理
- Vue Router 模块化路由
- Axios 请求封装
- mock / remote 模式切换
- 销售中心资源列表、详情抽屉、新增弹窗、设置占位

启动：

```bash
npm install
npm run dev
```
