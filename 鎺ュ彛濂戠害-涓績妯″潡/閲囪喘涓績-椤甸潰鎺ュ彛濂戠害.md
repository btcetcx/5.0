# 采购中心页面接口契约

适用模块：采购中心  
接口前缀：`/api/v1`  
覆盖页面：供应商管理、请购管理、询价管理、采购订单、采购退货、采购换货、采购报表

> 说明：供应商、请购、询价、采购订单是当前 5.0 原型中较完整的页面；采购退货、采购换货、采购报表在导航中已有入口，本文先定义首期契约，后续可按真实页面继续细化。

## 1. 模块范围

| 页面 | 前端文件参考 | 成熟度 | 主要能力 |
| --- | --- | --- | --- |
| 供应商管理 | `supplier-list.jsx` | 较完整 | 供应商列表、新增、详情、供货产品、联系人、财务、地址、报价记录、采购记录 |
| 请购管理 | `pr-list.jsx` | 较完整 | 请购列表、新增请购、请购详情、明细行、转询价、转采购 |
| 询价管理 | `inquiry-list.jsx` | 较完整 | 询价列表、新增询价、选择来源、供应商报价、定价、临时供应商 |
| 采购订单 | `order-list.jsx` | 较完整 | 采购列表、新增采购、选择来源、订单明细、金额、三单匹配、到票、付款、入库状态 |
| 采购退货 | 导航预留 | 预留 | 退货列表、新增退货、退货明细 |
| 采购换货 | 导航预留 | 预留 | 换货列表、新增换货、换货明细 |
| 采购报表 | 导航预留 | 预留 | 供应商对账、按供应商查询、按产品统计、退换货明细 |

## 2. 采购通用枚举

### 单据状态

| status | statusName | 适用 |
| --- | --- | --- |
| `draft` | 待提交 | 请购、询价、采购订单 |
| `pendingApproval` | 审批中 | 请购、采购订单、供应商 |
| `approved` | 已批准 | 请购、采购订单 |
| `rejected` | 已驳回 | 请购、采购订单 |
| `pendingInquiry` | 待询价 | 请购 |
| `transferredInquiry` | 已转询价 | 请购 |
| `pendingPurchase` | 待采购 | 请购 |
| `transferredPurchase` | 已转采购 | 请购 |
| `partialPurchased` | 部分采购 | 请购 |
| `inquiryInProgress` | 询价中 | 询价 |
| `inquiryCompleted` | 询价完成 | 询价 |
| `pendingPricing` | 待定价 | 询价 |
| `priced` | 已定价 | 询价 |
| `purchasing` | 采购中 | 采购订单 |
| `inTransit` | 运输中 | 采购订单 |
| `pendingInbound` | 待入库 | 采购订单 |
| `partialInbound` | 部分入库 | 采购订单 |
| `completed` | 已完成 | 通用 |
| `closed` | 已关闭 | 通用 |
| `voided` | 作废 | 通用 |
| `cancelledException` | 异常取消 | 采购订单 |

### 供应商状态

| status | statusName |
| --- | --- |
| `temporary` | 临时 |
| `pendingApproval` | 待审核 |
| `approved` | 已审核 |
| `disabled` | 已停用 |

### 供应商类型

| type | typeName |
| --- | --- |
| `rawMaterial` | 原材料供应商 |
| `parts` | 零部件供应商 |
| `package` | 包装供应商 |
| `service` | 服务供应商 |
| `temporary` | 临时供应商 |

## 3. 供应商管理

### 页面字段

列表字段：

| 字段 | 说明 |
| --- | --- |
| `code` | 供应商编号 |
| `name` | 供应商名称 |
| `typeName` | 供应商分类 |
| `categoryName` | 细分分类 |
| `contactName` | 主联系人 |
| `contactPhone` | 联系方式 |
| `statusName` | 供应商状态 |
| `source` | 来源 |
| `createdAt` | 建档日期 |
| `ownerName` | 采购负责人 |

详情字段分组：基础信息、联系人、地址、财务结算、供货产品、附件、报价记录、采购记录。

### 接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/api/v1/suppliers` | 供应商列表 |
| `POST` | `/api/v1/suppliers` | 新增供应商 |
| `GET` | `/api/v1/suppliers/{id}` | 供应商详情 |
| `PATCH` | `/api/v1/suppliers/{id}` | 编辑供应商 |
| `POST` | `/api/v1/suppliers/{id}/submit` | 提交审核 |
| `POST` | `/api/v1/suppliers/{id}/approve` | 审核通过 |
| `POST` | `/api/v1/suppliers/{id}/disable` | 停用供应商 |
| `GET` | `/api/v1/suppliers/{id}/products` | 供货产品 |
| `GET` | `/api/v1/suppliers/{id}/contacts` | 联系人 |
| `GET` | `/api/v1/suppliers/{id}/quotes` | 报价记录 |
| `GET` | `/api/v1/suppliers/{id}/purchase-records` | 采购记录 |

## 9. 需要和后端确认

- 请购转询价、转采购是否允许部分明细转换。
- 询价定价是否支持同一物料多供应商中标。
- 采购订单和仓储入库的来源关系字段是否统一为 `sourceId`、`sourceCode`、`sourceLineId`。
- 三单匹配的验收口径：订单、入库、发票是否首期都做。
- 采购退货、采购换货、采购报表首期是否只保留接口占位。
