# 质检中心交付记录

更新时间：2026-05-30

## 入口矩阵

本轮质检中心入口以 `vue3/src/layouts/erp-shell/navigation.ts` 当前质检导航为准，`qc-list.jsx` 仅作为字段、示例数据、页面动作和业务交互参考，未使用旧分类导航覆盖 Vue 导航。

| 二级导航 | 路由 | 分流 |
| --- | --- | --- |
| 工作台 | `/qc` | 保持工程默认工作台，不做质检私有工作台替换 |
| 质检管理 | `/qc/execution` | `action=new` 新增检验任务；`id=qc_001` 详情；浮窗动作按检验任务/待检/IQC/IPQC/FQC/OQC 做列表过滤 |
| 异常处理 | `/qc/exceptions` | `action=new` 新增异常；`id=ncr_001` 详情；不合格记录、隔离/拒收、返工复检、让步放行按异常分类过滤 |
| 质检分析 | `/qc/reports` | 质量趋势、不良分析、供应商质量、工序质量 |
| 质检设置 | `/qc/standards` | 标准配置列表/新增/详情；`setting=groups/fields/numbers/approvals/strategies/print` 进入设置母版 |

## JSX 严格继承项

- 状态集合：待报检、抽样中、待判定、特采评审、拒收退回、已放行；异常待处置、隔离中、复检中、让步审批、CAPA执行中、已关闭等。
- 动作集合：高级筛选、批量处理、导入、导出、新增、查看、复检、提交结果、让步放行、拒收/隔离、CAPA/8D、验证关闭、生成报告。
- 详情 Tab：质检信息、来源记录、抽样记录、检验明细、不良处置、复检记录、放行/拒收记录、质检报告；异常详情保留 CAPA/8D 与验证关闭。
- 检验明细字段：检验项目、方法、标准值、上下限、实测值、缺陷等级、结论、图片。
