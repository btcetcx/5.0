# 协作看板

## 当前阶段

Vue3 工程验收调整阶段。不要按旧 JSX 页面重做；以已完成 Vue3 工程、公共母版和最新验收意见为准。

## 协作流程

1. 每个人从 `main` 新建自己的功能分支。
2. 开发前先同步 `main`。
3. 每个 PR 只提交自己负责模块的改动。
4. 涉及公共组件、导航、路由、接口契约时，在 PR 描述中单独说明。
5. 合并前至少执行页面访问检查；能运行时执行类型检查或构建。
